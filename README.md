# DevDcii Portfolio

Personal portfolio with admin panel built with PHP and MySQL with Groc API.

## Tech Stack
- HTML, CSS, JavaScript
- PHP
- MySQL
- Cloudinary (media storage)

## Features
- Portfolio project showcase with media gallery
- Contact/inquiry form
- Admin panel (project management, inquiry tracking)
- Responsive design

## Local Setup
1. Clone the repo
   git clone https://github.com/devdcii/chan-portfolio.git

2. Set up your database
   - Create a MySQL database
   - Import the SQL schema

3. Configure database connection
   - Copy api/config.example.php to api/config.php
   - Fill in your database credentials

4. Run with XAMPP or any PHP server
   - Place project in htdocs
   - Access via localhost/devdcii
